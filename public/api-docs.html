Volleyball Analysis API Documentation

API endpoints for volleyball technique analysis and training

Home Google AI Demo OpenAI Agents Demo

Introduction

This API provides endpoints for analyzing volleyball techniques, generating training programs, and providing real-time feedback using artificial intelligence.

The API is divided into two main sections:

  • Google AI Integration: Endpoints for analyzing images, videos, and frames using Google's Gemini Pro Vision model.
  • OpenAI Agents Integration: Endpoints for comprehensive analysis, real-time feedback, training programs, and team analysis using OpenAI's Agents SDK.

Google AI Integration Endpoints

POST /api/volleyball/analyze-image

Analyze a volleyball image using Google's Gemini Pro Vision model.

Request Parameters (multipart/form-data)

Parameter Type Required Description
image File Yes The volleyball image to analyze
analysis_type String No Type of analysis to perform: "technique", "positioning", or "tactics" (default: "technique")

Response (application/json)

{
  "analysis": "Detailed analysis of the volleyball technique...",
  "timestamp": "2023-10-15T14:30:00Z"
}

POST /api/volleyball/analyze-video

Analyze a volleyball video by processing frames at regular intervals.

Request Parameters (multipart/form-data)

Parameter Type Required Description
video File Yes The volleyball video to analyze
analysis_type String No Type of analysis to perform: "technique", "positioning", or "tactics" (default: "technique")
interval_seconds Number No Interval between frames to analyze in seconds (default: 2)

Response (application/json)

{
  "analysis_results": [
    {
      "timestamp": "00:02",
      "analysis": "Detailed analysis of the volleyball technique..."
    },
    {
      "timestamp": "00:04",
      "analysis": "Detailed analysis of the volleyball technique..."
    },
    ...
  ],
  "summary": "Overall summary of the video analysis..."
}

POST /api/volleyball/analyze-frame

Analyze a single volleyball frame, useful for real-time analysis.

Request Parameters (multipart/form-data)

Parameter Type Required Description
frame File Yes The volleyball frame to analyze
analysis_type String No Type of analysis to perform: "technique", "positioning", or "tactics" (default: "technique")

Response (application/json)

{
  "analysis": "Detailed analysis of the volleyball technique...",
  "timestamp": "2023-10-15T14:30:00Z"
}

OpenAI Agents Integration Endpoints

POST /api/volleyball-agent/analyze

Analyze a volleyball video using the OpenAI Agents system.

Request Parameters (application/json)

Parameter Type Required Description
video_data String Yes Base64 encoded video data
player_data Object No Player information (playerId, name, position, skillLevel)

Response (application/json)

{
  "technique_analysis": [
    {
      "frame": 0,
      "analysis": "Detailed analysis of the volleyball technique..."
    },
    ...
  ],
  "positioning_analysis": [...],
  "tactical_analysis": [...]
}

POST /api/volleyball-agent/feedback

Get real-time feedback for a single video frame.

Request Parameters (application/json)

Parameter Type Required Description
frame_data String Yes Base64 encoded image data
current_time Number No Current time in the video (seconds)

Response (application/json)

{
  "feedback": "Immediate, actionable feedback on the volleyball technique...",
  "timestamp": 10.5
}

POST /api/volleyball-agent/training

Generate a personalized training program.

Request Parameters (application/json)

Parameter Type Required Description
player_id String Yes Player ID
technique_focus String No Specific technique to focus on (e.g., "spike", "serve")

Response (application/json)

{
  "player_id": "player123",
  "technique_focus": "spike",
  "program": {
    "title": "Personalized Spike Training Program",
    "description": "This program is designed to improve your spiking technique...",
    "drills": [
      {
        "name": "Warm-up Drill",
        "description": "Start with a 10-minute warm-up...",
        "duration": "10 minutes",
        "intensity": "Low"
      },
      ...
    ]
  }
}

POST /api/volleyball-agent/team-analysis

Analyze team performance.

Request Parameters (application/json)

Parameter Type Required Description
team_id String Yes Team ID
game_data Object No Game data for analysis

Response (application/json)

{
  "team_id": "varsity_2025",
  "analysis": {
    "strengths": [
      "Strong serving performance",
      "Effective blocking at the net",
      "Good communication between players"
    ],
    "weaknesses": [
      "Inconsistent passing in serve receive",
      "Defensive positioning needs improvement",
      "Transition offense is slow"
    ],
    "recommendations": [
      "Focus on serve receive drills in practice",
      "Work on defensive positioning and movement",
      "Practice faster transitions from defense to offense"
    ]
  }
}
